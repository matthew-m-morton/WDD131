Overview

Purpose

There is a game called Valheim and it is a crafting game that has lots of weapons and multiple boss enemies. There are also different playstyles to go along with the different weapons. This website will let players explore how effective their weapon of choice is against a boss.

Audience

My audience is the Valheim gaming community.

Dynamic elements

For my dynamic elements I plan to have the home/parent page explain what the website is and how it works. it will link to a child site which will be built with main containers that will be built from an array of objects. based oof the variable the user chooses, there will be calculations done and results displayed.
